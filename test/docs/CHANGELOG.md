# Changelog

## V3.3.0 TEST — Module Paramètres enrichi

### Nouveautés
- Ajout d'un bloc Déplacements dans Paramètres.
- Centralisation des kilomètres offerts, du tarif au km et du mode aller-retour.
- Ajout d'un exemple de calcul automatique des frais de déplacement.
- Ajout des modèles de mails pour devis et factures.
- Ajout d'un modèle de relance devis, préparé pour une future fonction de relance.
- Variables disponibles dans les modèles de mails : `{client}`, `{numero}`, `{type}`, `{montant}`, `{date}`, `{entreprise}`.
