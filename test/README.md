# L'Atelier Fleurs & Sens — V3.5.2 TEST MODULAIRE

Version test dédiée au bouton **Préparer mon rendez-vous** dans le module Mariages.

## Nouveautés V3.5.2

- Ajout d’un bouton **🎯 Préparer mon rendez-vous** dans **Clients > Mariages**.
- Formulaire guidé pour les rendez-vous téléphoniques mariage : contact, date, lieu, thème, couleurs, inspirations, fleurs aimées/à éviter, prestations souhaitées, contraintes et notes internes.
- Création automatique d’une fiche mariage à partir du rendez-vous.
- Génération automatique :
  - des articles souhaités,
  - de la synthèse du projet,
  - des besoins / mémo,
  - de deux tâches de suivi : envoyer la synthèse et préparer le devis.
- Possibilité de pré-remplir l’assistant depuis une fiche mariage existante avec **Préparer / compléter RDV**.

## Conservé depuis V3.5.0

- Wedding Manager dans chaque fiche mariage.
- Progression, timeline, jalons J-, todo, documents et budget.
- Suivi mariage avec cases automatiques et manuelles.
- Mariages en cours triés par date de livraison.
- Prestations complémentaires mariage et atelier.
- Tableau de bord V3.2.1.
