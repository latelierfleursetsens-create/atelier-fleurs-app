# MyBusiness — V4.0.9 PROD

## Nouveauté

- Ajout d’un champ **Tarif par personne (€)** dans les ateliers de catégorie **Atelier thématique site**.
- Le montant de référence est calculé automatiquement : nombre de participants prévu × tarif par personne.
- Ce montant reste affiché uniquement dans **Finances**.
- Lors de l’ajout manuel d’une participante, son montant est prérempli avec le tarif par personne enregistré.

## Test conseillé

Ouvrir un atelier thématique site, saisir 10 participants et 35 € par personne, enregistrer, puis vérifier dans Finances que le montant de référence est de 350 € avant déduction des encaissements.
